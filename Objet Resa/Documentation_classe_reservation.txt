-LA CLASSE RESERVATION-

La classe resa permet d'interagir avec la table 'Reservation' présente
dans la base de données. Elle contient donc les mêmes variables que la table
de la base de données.

I) Création de l'objet

	Pour créer un objet de type 'resa', vous devez avoir préalablement
	créé un objet de type 'PDO' permettant l'accès à la base de données.
	Pour la suite, l'objet PDO portera le nom $dbh.
	
	Pour créer un objet 'resa', vous devez instancier un nouvel objet en lui fournissant les variables :
            - dbh
            - date de début
            - date de fin
            - id du locataire (facultatif)
            - validation (0 pas validé, 1 validé)
        Ce qui donnera $NomDeVotreObjet = new resa(liste des paramètres)
      
	qui va implementer les instances de l'objet et verifier si ces données
	sont présentes dans la base de données. Si elle le sont, l'id de la ligne
	sera mit dans l'instance $id de l'objet créé.
	
II) Accesseurs 

	Les getters ont un nom suivant la même structure, get_NomDeLaVariable() :
	par exemple pour $id : get_id()
	Il en est de même avec les setters, set_NomDeLaVariable($NomDeLaVariable) :
	par exemple pour $id : set_id($id)
	
III) Interaction avec la base de données

	1) création dans la base de données

		Pour ajouter votre objet dans la base de données, il faut appeler la
		fonction 'creation' prennant en paramètre l'objet 'PDO' utilisé lors
		de la création de votre objet 'locataire' :
		creation($dbh)
		
		Si vos valeurs sont déjà présentent dans la base de données, la fonction
		vous le dira via une sortie écran (echo).
	
	2) modification dans la base de données
		
		Pour modifier les valeurs de votre objet dans la base de données, il
		faut utiliser la fonction 'modification' prenant en paramètre l'objet 'PDO'
		($dbh) et les instances de l'objet ($date_deb,$date_fin,$id_locataire,$validation) :
		modification($dbh,$date_deb,$date_fin,$id_locataire,$validation)
		
		Si, parmi les paramètres, il y en a un dont la valeur ne doit pas être changé,
		faites un appel à la fonction get de la valeur. Exemple avec la modification
		du mail mais pas de nom et prénom : 
		modification($dbh,get_nom(),get_prenom(),$maNouvelleAdresseMail)
		
		Si les valeurs de votre objet ne sont pas présentes dans la base de données,
		la fonction vous le dira via une sortie écran (echo)
		
	3) suppression dans la base de données
		
		Pour supprimer les valeurs de l'objet dans la base de données, il suffit
		d'utiliser la fonction 'destruction' prenant en paramètre l'objet PDO ($dbh):
		destruction($dbh)
		
		Si les valeurs de votre objet ne sont pas présentes dans la base de données,
		la fonction vous le dira via une sortie écran (echo)	

        4) Validation d'une réservation
                
                Appel de la fonction validation($dbh).
